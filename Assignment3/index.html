dTree Library

We used the Library dTree for the chart, because the library is relatively easy to use and offers the possibility to visualize trees that have more than one parent
Normal D3 trees do not support this feature of course.
Another advantage of this specific library is, that it also has the possibility for someone to have more than one spouse, which most of the other libraries we looked at did not support.
Also, dTree is based on D3 v4, which makes it very useful for the Interactive Data Exploration Course
