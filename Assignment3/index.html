    Trump Family Tree


Figure 1

Figure 1 - dTree Library

We used the Library dTree for the chart, because the library is relatively easy to use and offers the possibility to visualize trees that have more than one parent.
Normal D3 trees do not support this feature of course.
Another advantage of this specific library is, that it also has the possibility for someone to have more than one spouse, which most of the other libraries we looked at did not support.
Also, dTree is based on D3 v4, which makes it very useful for the Interactive Data Exploration Course

Since family tree is not a normal tree, we chose to use dTree library instead of d3.
Because there are two parents to each set of children and sometimes people have multiple partners, it was difficult to create the links between the right nodes.

    Visualization


Improvements

Since the Trump family is quite complex, we thought the best way to differentiate relationships is by color.
Marriage is defined by darkblue, while for children we used lightblue.
Given the 2 colors, it is much easier to track the connection between the nodes.


For distinguishing the persons more easier, pink boxes were used for women and blue for men


Regarding improvements, the visualization would be a lot much interactive if we would have added dynamic elements.
For example, instead of the boc with the name, an avatar of each member could be used.
While hovering on the picture, a description box would pop up containing details about that person.
